# Print Preview Studio

Local web app for visualizing print output. Upload a document, select duplex or simplex, choose flip side, set copies, and preview a 3D stack of the printed sheets.

## Requirements

- Bun

## Getting started

- Install dependencies: bun install
- Run the dev server: bun run dev
- Build for production: bun run build
- Preview production build: bun run preview
